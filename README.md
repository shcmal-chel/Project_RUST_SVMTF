
# 🚦 Система визуального моделирования транспортных потоков

## 📋 О проекте

Веб-приложение для моделирования транспортных потоков, разработанное на **Rust** с компиляцией в **WebAssembly**. Приложение позволяет визуализировать движение транспорта, анализировать загруженность дорог и тестировать различные сценарии дорожного движения прямо в браузере.

### Основные возможности

- **Дорожная сеть**: Крестообразный перекресток (2 дороги, 2 въезда, 2 выезда).
- **Светофоры**: Динамическое переключение с настраиваемым циклом.
- **Транспорт**: 3 типа транспортных средств (легковые автомобили, грузовики, автобусы).
- **Статистика**: Мониторинг загрузки дорог, средней скорости и пропускной способности.
- **Сценарии**: 4 предустановленных режима симуляции.
- **Отчеты**: Экспорт результатов моделирования в текстовый файл.
- **Управление картой**: Масштабирование (zoom) и панорамирование (pan).

## 🚀 Установка и запуск

### Требования

- [Rust](https://www.rust-lang.org/tools/install) версии 1.70+
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)
- [Node.js](https://nodejs.org/) (для запуска локального сервера)

### Установка зависимостей

```bash
# Установка Rust (если не установлен)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Установка wasm-pack
cargo install wasm-pack
```

## Запуск проекта

1. Клонирование репозитория:
    ```bash
    git clone https://github.com/shcmal-chel/Project_RUST_SVMTF.git
    cd traffic_simulation
    ```
2. Сборка WebAssembly модуля:
    ```bash
    wasm-pack build --target web --out-dir web/pkg -- --no-default-features
    ```
3. Запуск локального сервера:
    ```bash
    cd web
    npx http-server -p 8000
    ```
4. Откройте приложение в браузере:
    👉 http://localhost:8000

## 📊 Сценарии

Приложение поддерживает 4 режима симуляции, которые можно переключать кнопками **1-4**.

### 1. Базовое движение
*   **Описание:** Нормальная интенсивность движения.
*   **Параметры:**
    *   🟢 Зеленый свет: 35 сек
    *   🔴 Красный свет: 30 сек
    *   Интенсивность спавна: `0.3`
    *   Цикл светофора: 70 сек

### 2. Увеличение интенсивности
*   **Описание:** Поток транспорта увеличен в 3 раза. Возможны заторы.
*   **Параметры:**
    *   Интенсивность спавна: `0.9`
    *   Визуализация: Цвет дорог меняется на желтый/красный при высокой загрузке.

### 3. Перекрытие дороги
*   **Описание:** Западная дорога перекрыта.
*   **Параметры:**
    *   Движение возможно только по вертикальной дороге.
    *   Визуализация: Красная подсветка проблемной зоны.

### 4. Оптимизация светофоров
*   **Описание:** Ускоренный цикл светофоров для разгрузки перекрестка.
*   **Параметры:**
    *   🟢 Зеленый свет: 20 сек
    *   🔴 Красный свет: 18 сек
    *   Цикл светофора: 41 сек (на ~40% быстрее базового)

## 📄 Отчеты

При нажатии кнопки **"📄 Отчет"** формируется детальный текстовый отчет о текущем состоянии симуляции, который можно сохранить на диск.

**Содержание отчета:**
*   Дата и время симуляции.
*   Общее количество созданных транспортных средств.
*   Количество активных ТС в данный момент.
*   Средняя скорость движения.
*   Пропускная способность перекрестка.
*   Загрузка каждой дороги (в процентах).
*   Список активных ТС с их текущим прогрессом пути.

## 🔧 Разработка

### Добавление нового сценария

Чтобы добавить новый сценарий, отредактируйте файл `src/lib.rs`:

1. Найдите функцию `load_scenario`.
2. Добавьте новый вариант в match-выражение:

```rust
4 => {
    // Пример существующего сценария
    self.scenario_name = "Оптимизация".to_string();
    self.light_cycle = 41.0;
    self.light_green1 = 20.0;
},
5 => {
    // Ваш новый сценарий
    self.scenario_name = "Новый сценарий".to_string();
    self.light_cycle = 50.0;
    self.light_green1 = 25.0;
    
    // Изменение параметров спавна
    for entry in &mut self.network.entry_points {
        entry.spawn_rate = 0.5;
    }
}
```

### Изменение параметров дорог 

Для изменения геометрии дорог отредактируйте файл `src/traffic_network/mod.rs`:

```rust
let road1 = RoadSegment {
    start: Point { x: 10.0, y: 50.0 },
    end: Point { x: 90.0, y: 50.0 },
    length: 80.0,
    // ... остальные параметры
};
```
